# Arabic Dialect Classification

This project focuses on classifying Arabic text into dialects using both traditional machine learning and deep learning approaches.

## Overview
In this project, I built and compared two models to classify Arabic dialects:
- Logistic Regression with TF-IDF
- LSTM (Deep Learning)

## Results
- Logistic Regression Accuracy: 88.2%
- LSTM Accuracy: 93.3%

The LSTM model outperformed Logistic Regression, showing better ability to capture contextual patterns in Arabic text.
